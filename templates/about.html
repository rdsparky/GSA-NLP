About Application.

This application is based on the concept of natural language processing and implemented using machine learning algorithms.

The applications takes an input review from user(must be text)and generates result stating whether the input is positive or negative.

The confidence percent here is the percentage of certainty of our model(i.e. how much it is certain that the result generated is correct).
